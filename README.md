# Dot challenge: TMDB films
- Integration with tmdb api, build an SPA with checkout page


## Technologies

- Vue
- Pinia
- Tailwind
- Vue Router
- Axios

Work in progress

## Setup and Run
- clone/download the repo
- navigate to 'dot-films' dir
- npm install && npm run dev
- open your browser on localhost:3000
- have fun :)
